How to share code snippets from my project?

To share a code snippet, open the file with the content you would like to share, in the editor.

Make sure the file has some content in it, or the share code function won't work.

Select the share icon from the toolbar on the top, you'll be prompted for a title for your snippet.

This title needs to be unique since it will be used in creating a permalink for your snippet.

On successful snippet creation, you'll see an alert with the sharable link to your newly shared snippet. You can also go the blog to check out snippets shared by others.
