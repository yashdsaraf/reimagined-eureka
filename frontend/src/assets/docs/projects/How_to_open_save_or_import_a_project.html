How to open, save or import a project?

  • How to open a project?

    From the home page select the Open project option.

    You can open a project saved offline or online.

    The app will then verify whether the project with the same UUID exists in the database.

    On successful verification the settings will be auto imported and you will be redericted to the editor.


  • How to save a project?

    When a project is currently open, you can save the project by clicking the save icon from the toolbar.

    You can either save a project online or offline.


  • How to import a project?

    From the home page select the Open project option.

    You can import a project saved offline or online.

    The app will extract the project file and ask you for the new settings.
