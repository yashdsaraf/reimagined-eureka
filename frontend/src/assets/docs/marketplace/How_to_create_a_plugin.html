How to create a plugin?

To create a plugin, a user needs to have developer permissions.

For more info, see: https://www.youtube.com/watch?v=Skk-fNV-D5M

Once logged in as developer, go to the developer console and click add plugin

.

After filling out all the plugin details, you'll be asked to decrypt an otp text file using your private key.

On entering the correct otp, your new plugin will get created.

Only approved plugins will be visible in the marketplace.

See also: https://www.youtube.com/watch?v=aixbKGuTMtY
