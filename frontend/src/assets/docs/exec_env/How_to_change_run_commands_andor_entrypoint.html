How to change run commands and/or entrypoint?

Run commands or entrypoint can be changed from the settings menu in the editor page.

To do so, click the cog icon in the bottom right of the index page and select the Edit run commands button.

Here you can change the run commands and entrypoint for your current project or reset the run commands back to their original state.

Be very careful when changing the run commands as they might have unwanted effects on your work.
